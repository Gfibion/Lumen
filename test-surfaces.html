Test 03 Named Materials — define once, use anywhere

Two reusable materials defined at the top. Three surfaces apply them via use:. Any surface using the same material name receives identical styling.

LUMEN source
// Named material definitions
material panel-dark {
  color:   #111827
  radius:  10px
  border:  1px solid rgba(255,255,255,0.06)
  shadow:  0 4px 24px rgba(0,0,0,0.4)
}

material panel-accent {
  color:   #1e3a5f
  radius:  10px
  border:  1px solid rgba(59,130,246,0.3)
}

// Three surfaces — two materials shared
surface row {
  width:  100%
  layout: row
  gap:    12px

  surface a {
    width:  fill
    height: 90px
    use:    panel-dark
  }
  surface b {
    width:  fill
    height: 90px
    use:    panel-accent
  }
  surface c {
    width:  fill
    height: 90px
    use:    panel-dark
  }
}
Rendered output
material definition · use: · color property in material

Test 05 Layout System — row, column, fill, gap, align, justify

A two-column layout where the sidebar has a fixed width and the main area uses fill to consume remaining space. Children in both columns are stacked vertically.

Test 06 Glass Material — backdrop-filter, border, shadow

  justify: center
